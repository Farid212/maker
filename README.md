# Maker

**Maker** is a To Do mobile application created with React Native.

## Basic features  
  
- Task list
- Create, Edit and Remove task

## Technologies  
**Maker** was created together with:  
  
- React <i>16.5.0</i>
- React Native <i>32.0.0</i>
- React Redux <i>6.0.0</i>  
- Redux <i>4.0.1</i>  
- Redux Thunk <i>2.3.0+</i>  
- Axios <i>18.0+</i>
